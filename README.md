# hyderabad-aqi-gemini-advisory
AQI forecasting for Hyderabad with Gemini-powered bilingual public advisories
Hyderabad's air quality data exists through CPCB monitoring stations, but it's raw and technical — not something a resident, parent, or asthma patient can act on before the air turns bad. This solution forecasts 7-day-ahead AQI across 5 Hyderabad monitoring stations (Bollaram, Zoo Park, ICRISAT Patancheru, IDA Pashamylaram, Central University) using an XGBoost model trained on 30-day historical pollutant windows (PM2.5, PM10, NO2, SO2, O3, CO).
The key Gen AI layer sits on top: the Gemini API takes each station's structured forecast (predicted AQI, dominant pollutant, trend) and generates a short, plain-language public health advisory — automatically translated into natural, conversational Telugu alongside English — so residents get an actionable warning in the language they actually speak, not a raw number on a dashboard. Google AI Studio was used to prototype and refine the advisory prompt for tone and clarity before integrating it into the pipeline.
In testing, the system correctly distinguished an industrial hotspot (Bollaram, AQI 318, "Very Poor") from residential zones (Central University, AQI 168, "Moderate") in the same forecast run — producing area-specific advisories rather than one generic city-wide alert.
